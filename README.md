# Laundry Mart Webpage

## Description
This project is a simple HTML webpage for Laundry Wallah or Laundry Mart. It presents information about a laundry service including description, services offered, pricing details, and a basic booking form.

## Project Structure
project-folder/
index.html  
README.md  

## How to Run
1. Double click on `index.html` or open it using any web browser.

## Features
- Displays a welcome heading and introduction about the laundry service
- Shows an image related to laundry
- Lists all available services
- Provides a price table for different services
- Includes a simple booking form for users
- Shows contact information at the bottom

## Technologies Used
HTML5

## Output
When the file is opened in a browser, it shows a complete laundry service webpage with:
- Service description
- Services list
- Pricing table
- Booking form
